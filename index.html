Flightgroove

flightgroove is a small flight explorer I started building on the skyscanner API recently, to try out some ideas like multiple-origin flight search, and integration of destination cost-of-living data.

It's built as a SPA in Elm, a compile-to-js functional programming language. The backend API is built using NodeJS, with Mocha for tests.

Check it out on github

Type Scale Designer

A prototype I used for messing around with Angular 1 back in 2014, the idea behind this was to help backfit an existing set of font sizes in a website, onto a modular scale.

If you enter the font sizes currently used, the app tries to map existing styles onto their closest equivalents in a scale. Allowing you to clean up your type proportions without making any drastic changes.

Check it out on github
